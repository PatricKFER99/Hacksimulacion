#  Ghost-Trace Simulator

![Status](https://img.shields.io/badge/Status-Educational-blue) ![License](https://img.shields.io/badge/License-MIT-green)

**Ghost-Trace** es una herramienta de simulación visual basada en la web diseñada para recrear interfaces de hacking cinematográficas (estilo *Mr. Robot* o *Matrix*).

Este proyecto está diseñado con fines **recreativos, educativos y de entretenimiento** (bromas, filmaciones, demos de UI). **NO** realiza ninguna conexión real ni extrae datos privados. Todos los datos mostrados son generados aleatoriamente en el lado del cliente (Client-Side).

##  Características

- **Efecto Matrix:** Lluvia de código binario en HTML5 Canvas.
- **Terminal Interactiva:** Simulación de línea de comandos.
- **Datos Aleatorios:** Generación procedural de nombres, IPs y ubicaciones falsas.
- **Diseño Responsivo:** Adaptable a pantallas completas para filmación.

##  Tecnologías

- **HTML5** (Estructura Semántica)
- **CSS3** (Animaciones CRT, Glow Effects)
- **JavaScript ES6+** (Lógica de simulación y Canvas)
